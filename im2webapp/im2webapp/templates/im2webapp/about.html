{% trans "About" %}

O projeto IM2WebApp teve como objetivo o desenvolvimento de uma plataforma para auxílio ao ensino de processamento de images médicas.

Foi iniciado por Rodolfo dos Santos Ribeiro que apresentou uma interface em MATLAB para processamento de imanges.

Em continuidade, Ronaldo Sena reviveu o projeto na disciplina de imagens meédicas 2 do curso de Engenharia Biomédica da Universidade Federal de Uberlândia.

Seguindo com o apoio da nossa orientadora, Ana Claudia Patrocinio, o projeto foi implementado como uma aplicação web pelo aluno Italo Gustavo Sampaio Fernandes, como seu TCC.
